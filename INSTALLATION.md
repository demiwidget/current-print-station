# Quick Installation Guide

## Install the Chrome Extension

### Step 1: Download
- Download the `current-rms-print-server-extension.zip` file
- Extract it to a folder on your computer

### Step 2: Load in Chrome
1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right)
3. Click "Load unpacked"
4. Select the extracted `current-rms-print-server-extension` folder
5. Pin the extension by clicking the puzzle piece icon and pinning "Current-RMS Print Server"

### Step 3: Configure
1. Click the extension icon in your toolbar
2. Enter your Current-RMS subdomain (without .current-rms.com)
3. Enter your Current-RMS API key
4. Click "Save Configuration"
5. Click "Test Connection" to verify

### Step 4: Use
1. Open a job in Current-RMS
2. Click the extension icon
3. Scan or type a barcode in the search field
4. Preview and print the matching label
